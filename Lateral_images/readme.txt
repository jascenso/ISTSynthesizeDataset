Lateral_images Folder - Contains the lateral views used to synthesize new views with VSIM
	
	Each sequence folder contains a folder with texture and depth lateral views
	
	DEPTH folder:
		x_depth_QPyVw_z means depth map of x video sequence, QP equal to y, input view w and frame z
		depth_x_w_z means depth map of x video sequence, QP equal to 0, input view w and frame z
	
	TEXTURE folder:
		x_texture_QPyVw_z means texture view of x video sequence, QP equal to y, input view number w and frame z
		xw_z means texture view of x video sequence, QP equal to 0, input view number w and frame z
